# divvy-pop
App to create web sessions, scan receipt or bill, invite others to claim items to split the bill.

# Project Divvy

## Application Context

This is my first attempt at building an application. The concept, description, prototype mockups, etc. are all created by me. The code itself is developed via Claude Code.

My goal is to **learn the code by seeing the finished product**, though I recognize my career path isn’t likely in software engineering.

My second goal is to **build something that is sleek, effective, and usable** so that I can follow through on a project from concept to completion.

## Project Context

I find that it is particularly annoying to go to a group outing (like a dinner), receive a single check, and then have to pass it around while we sort out who owes how much and how folks should divvy up the appetizers or shared plates. The idea here is to solve this with a simple web app.

The ideal finished output:
1. Start a session
2. Scan the receipt/check/bill
3. Generate a QR code and shareable link
4. Share the code and/or link to others
5. They join the session with their name
6. Everyone “claims” their items and sees the total they owe.
7. Users can then copy their amount and jump to their preferred money-sending app (e.g. Venmo, Cash, PayPal, Apple Pay, etc.) to finish.

The expectation for the foreseeable future is that this project will not be intended for monetization purposes.
